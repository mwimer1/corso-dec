# Icon

The icon directory contains UI components for the Corso platform.

UI components for the components system, following atomic design principles. Located in ui/atoms/icon/.

## Directory Structure

```
components/ui/atoms/icon/
icon-base.tsx
icons/
  icons/building-icon.tsx
  icons/hamburger-icon.tsx
  icons/index.ts
  icons/mail-icon.tsx
  icons/message-square-icon.tsx
  icons/up-arrow-icon.tsx
  icons/user-icon.tsx
  icons/x-mark-icon.tsx
```

## Usage

Import components from the appropriate subdirectory:

```typescript
import { ComponentName } from '@/components/ui/atoms/icon/subdirectory';
```
