import { Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter } from '@/organisms/dialog';

# Dialog

A composable modal dialog built on Radix Dialog with tokenized overlay/content variants and helper subcomponents.

## API

Exports:
- `Dialog` (Root)
- `DialogTrigger`
- `DialogContent` (props: `size?: DialogSize`)
- `DialogHeader` `DialogFooter`
- `DialogTitle` `DialogDescription`

## Usage

```tsx
<Dialog>
  <DialogTrigger asChild>
    <Button>Open</Button>
  </DialogTrigger>
  <DialogContent size="md">
    <DialogHeader>
      <DialogTitle>Modal title</DialogTitle>
      <DialogDescription>Short description.</DialogDescription>
    </DialogHeader>
    …body…
    <DialogFooter>
      <Button variant="outline">Cancel</Button>
      <Button>Confirm</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

## Keyboard Matrix

- Arrow keys: handled by content (no special bindings)
- Enter/Space: activate focused actionable
- Escape: closes dialog (Radix default)
- Focus trap and aria attributes are handled by Radix.

## Accessibility

- Title/Description map to proper roles via Radix
- Overlay uses tokenized styles and respects reduced motion
