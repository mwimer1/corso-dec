# Marketing Page Sections

## Purpose

Reusable section components for marketing pages: pricing, contact forms, legal content, etc.

## Key Files

- Individual section components in subdirectories

## Usage

Used in `app/(marketing)/` routes to build marketing page content.

## Styling

- Uses Tailwind CSS with marketing design tokens
- Full-width sections with background variants
- Responsive layouts

## Client/Server Notes

- Sections are client components (interactive elements)
